O código foi utilizado com o intuito de identificar os frutos maduros e os frutos ainda verdes afim de separálos. A webcam foi 
aclopada em cima da esteira e conforme os frutos iam rolando, o programa os identificava e mandava uma resposta para um motor,
dessa forma os frutos maduros eram destinados para um lado e os verdes para outro. 
O código utiliza a biblioteca OpenCV e foi programado na idle do Python
